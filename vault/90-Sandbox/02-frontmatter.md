# 02 Frontmatter サンプル

このページは frontmatter に以下を含みます:

- `title`: ページタイトル
- `tags`: 階層タグ含む
- `aliases`: 別名での到達
- `date`: 作成日
- `author`: 著者
- `draft`: 下書きフラグ（false のため公開対象）
- `description`: ページ説明

## 検証ポイント

- 両 SSG で `title` が使われているか
- aliases での到達性は [[18-aliases|18]] を参照
